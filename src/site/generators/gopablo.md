## TL;DR

### Install GoPablo

```
sudo npm i gopablo -g
```

### Initialize project

Run GoPablo to generate the file structure:

```
gopablo
```

### Run the app in development mode

Build and open your browser to http://localhost:3000.

```
npm run dev
```

### Build distribution files for the app in production

```
npm run prod
```
